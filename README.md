## DJANGO
### How to start django
1.  Installing Django: 
`pip3 install Django==3.1.1`
2.  Creates project lecture3: 
`django-admin startptoject lecture3`
3.  Creates application part called hello: 
`python manage.py startapp hello`
4.  Runs server: 
`python .\manage.py runserver`
